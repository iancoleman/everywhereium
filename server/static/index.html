A way to tip content creators on the web

Add a 'tip' property to a div tag or the body tag on your page.
<div tip="your details here">Your content</div>
This lets everyone on the internet know how they can send you tips.
See what the tip property should include and generate a tip tag for your elements.

Key Features

  • Use any payment method - currently the everywhereium extension supports bitcoin and paypal.
  • Supports multiple tips per page - each indivdual content contributor can be tipped; works well on forums and crowd-sourced content.
  • Easily expanded - json format has been designed for flexibility so other payment methods can easily be included, even before they're supported in the browser extensions.
  • Open source - secure and community-driven

Giving Tips

  • Install the extension (currently chrome only, others are in the works)
  • If you like something on a page, press the extension button.
  • The extension looks for tip data, and presents a simple dialog.
  • Use the dialog to send a tip directly to the person that created the content you like.

FAQ

  • Why does this page look so horrid?
    • Everywhereium is looking for a person to do some styling. If you can create some graphics and do some css, please visit the project at github.com and have a look at the issues list, particularly the issue about style. Of course, tip data for contributors will be included on any page they contribute to.
  • When will extensions be available for browsers other than chrome?
    • When they get written! If you know code, please consider contributing to the project at github.com There's also opportunity to create javascript libraries and wordpress plugins etc that can also tap into the tip property. See the issues about the Firefox extension, the Internet Explorer extension, and the javascript library.
  • What if I want to tip a page that has no tip data on it?
    • Everywhereium is investigating an escrow service so you can still send a tip to that page, and the owner of the page can claim it back when they've added the required tip information to their page. If you have a good idea, please make a comment in the forum thread or the github issue.
  • How can I contact everywhereium?
    • The best means of contacting everywhereium is by email. ian@everywhereium.com
